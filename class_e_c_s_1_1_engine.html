Detailed Description

The heart of the Entity framework. It is responsible for keeping track of Entity and managing EntitySystem objects. The Engine should be updated every tick via the update(float) method. Supports Entity and Component pooling. This improves performance in environments where creating/deleting entities is frequent as it greatly reduces memory allocation.

With the Engine you can:

  • Create entities using createEntity()
  • Create components using createComponent()
  • Add/Remove Entity objects
  • Add/Remove EntitySystems
  • Obtain a list of entities for a specific Family
  • Update the main loop
  • Register/unregister EntityListener objects
