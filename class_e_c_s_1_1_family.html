Detailed Description

Represents a group of Components. It is used to describe what Entity objects an EntitySystem should process. Example:

Family.all(PositionComponent.class, VelocityComponent.class).get()

Families can't be instantiated directly but must be accessed via a builder ( start with Family::all(), Family::one() or Family::exclude() ), this is to avoid duplicate families that describe the same components.
