Detailed Description

The heart of the Entity framework.

It is responsible for keeping track of Entity and managing EntitySystem objects. The Engine should be updated every tick via the update(float) method. Supports custom MemoryManager.

With the Engine you can:

  • Create entities using createEntity()
  • Add/Remove Entity objects
  • Add/Remove EntitySystems
  • Obtain a list of entities for a specific Family
  • Update the main loop
  • Connect to/Disconnect from EntitySignal
