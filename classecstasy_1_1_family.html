Detailed Description

Represents a group of Components.

It is used to describe what Entity objects an EntitySystem should process. Families can't be instantiated directly but must be accessed via a builder. This is to avoid duplicate families that describe the same components Start with Family::all(), Family::one() or Family::exclude().
